Workflow

Built to swap a mock formatter with Claude (or any LLM) later without rewriting the UI.

1. Capture

Paste rambles or summaries into the Projects page. Data stays local (localStorage) for offline work.

2. Shape

The mock formatProjectWithAI organizes content into title, description, tech, challenges, outcomes, and media placeholders.

3. Render

Projects appear as collapsible, engineering-style cards. Nothing overwrites—each intake is preserved.

4. Extend

Swap the mock formatter with a Claude API call inside assets/js/app.js when ready.

Design principles

  • Pure HTML/CSS/JS
  • No servers. No auth.
  • GitHub Pages compatible
  • Local-first; works offline
  • Engineering tone, minimal flair
